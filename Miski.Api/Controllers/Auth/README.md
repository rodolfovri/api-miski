# Módulo de Autenticación

Este módulo maneja todo lo relacionado con la autenticación y autorización de usuarios.

## Funcionalidades principales:
- Login/Logout de usuarios
- Registro de nuevos usuarios
- Cambio de contraseñas
- Validación de tokens
- Gestión de perfiles de usuario

## Controladores:
- `AuthController`: Operaciones de autenticación y gestión de usuarios

## Casos de uso implementados:
- Login de usuario
- Registro de usuario
- Obtener perfil del usuario autenticado
- Cambiar contraseña
- Validar token
- Cerrar sesión

## DTOs utilizados:
- `LoginDto`
- `RegisterDto`
- `AuthResponseDto`
- `ChangePasswordDto`
