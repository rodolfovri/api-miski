# Módulo de Personas

Este módulo gestiona todo lo relacionado con personas, usuarios y sus categorías.

## Funcionalidades principales:
- CRUD de personas
- Gestión de categorías de personas
- Asignación de categorías a personas
- Gestión de roles y usuarios

## Controladores:
- `PersonasController`: CRUD de personas y gestión de categorías
- `CategoriaPersonaController`: CRUD de categorías de personas

## Casos de uso por implementar:
- Crear/actualizar/eliminar personas
- Crear/actualizar/eliminar categorías
- Asignar/remover categorías a personas
- Consultar personas por categoría

## Entidades relacionadas:
- `Persona`
- `CategoriaPersona`
- `PersonaCategoria`
- `Usuario`
- `Rol`

## Tipos de personas:
- Proveedores
- Comisionistas
- Personal interno
