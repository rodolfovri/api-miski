# Módulo de Maestros

Este módulo gestiona las tablas maestras del sistema.

## Funcionalidades principales:
- CRUD de productos
- CRUD de tipos de documento
- CRUD de vehículos
- Gestión de stock por producto

## Controladores:
- `ProductosController`: CRUD de productos y stock
- `TipoDocumentoController`: CRUD de tipos de documento
- `VehiculosController`: CRUD de vehículos

## Casos de uso por implementar:
- Crear/actualizar/eliminar productos
- Consultar stock por producto
- Crear/actualizar/eliminar tipos de documento
- Crear/actualizar/eliminar vehículos

## Entidades relacionadas:
- `Producto`
- `TipoDocumento`
- `Vehiculo`
- `Stock`

Este módulo es fundamental para el funcionamiento básico del sistema.
